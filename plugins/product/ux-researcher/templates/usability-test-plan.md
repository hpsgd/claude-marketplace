## Moderator Guide

**Introduction**: "We are testing the [product/feature], not you — there are no wrong answers. Think aloud as you work through each task. The session will take about {{n}} minutes."

**Task delivery**: Read the scenario aloud, confirm understanding, start the timer on first action.

**Follow-up probes**: "What did you expect to happen?", "Was anything confusing?", "Where would you go next to {{related action}}?", "How confident are you that you completed the task?"

**Debrief questions**: Which task was easiest/hardest and why? What one thing would you change? How would you describe this product to a colleague?
