Write onboarding content for $ARGUMENTS. Focus on the fastest path from signup to genuine value. Structure: welcome (what the product does for THEM) → first action (single most important thing) → aha moment (guide to real value) → next steps. Each step < 2 minutes. Show, don't tell — use the product to teach the product. Progressive disclosure — don't show everything at once.
